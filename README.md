# 3. コア機能
- 会員登録/ログイン機能
  - 管理者が登録許可しなければログインできないように実装
- 投稿機能
  - quillスマートエディタを使用して多数の画像、ファイル添付可能
  - 管理者だけ投稿できるように実装
- 管理者機能
  - 委員会、委員会メンバー管理
  - 一般会員入会手続き
